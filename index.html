Privacy Policy

Last Updated: January 30, 2025

This privacy policy applies to the Pinterest API integration used by Cadence&court for content management purposes.

Information We Collect

This application accesses only the Pinterest account belonging to Cadence&Court for the sole purpose of creating and managing pins. We do not collect, store, or process any personal information from third-party users.

How We Use Pinterest API

  • Create and schedule pins to our boards
  • Access our board and pin analytics
  • Manage our board organization
  • All usage complies with Pinterest's API Terms of Service

Data Storage and Security

  • Authentication tokens are stored securely and used only for authorized API requests
  • No user data is shared with third parties
  • No tracking or analytics of other Pinterest users
  • All data handling follows industry-standard security practices

Third-Party Services

This application may use the following third-party services:

  • Make.com - Automation platform for workflow management
  • Anthropic Claude API - Content generation assistance
  • Canva - Design and graphics tool

All third-party services comply with their respective privacy policies and terms of service.

User Rights

As this application only accesses our own Pinterest account and does not collect third-party user data, there are no external user rights to manage. We maintain full control over our own data and API usage.

Changes to This Policy
